# github-wrapper

> My luminous Nuxt.js project

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

## 環境変数設定

### dotenvの設定

```
USER_NAME=
USER_PASSWORD=
```
